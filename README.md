# C-Chat-app
A simple chat application in c with file handling
# Feautures
-  Two-user chat system  
-  Saves complete chat history to `chat_history.txt`  
-  Clean and simple console interface  
-  Turns switch automatically between User 1 and User 2  
-  Type **exit** anytime to stop chatting  
-  Beginner-friendly C code

## How It Works
1. The program asks for:
   - Name of User 1  
   - Name of User 2  

2. Users type messages one by one.
